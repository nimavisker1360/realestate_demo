# راهنمای راه‌اندازی سیستم تولید خودکار بلاگ با هوش مصنوعی

## معرفی سیستم

سیستم تولید خودکار بلاگ با استفاده از هوش مصنوعی GPT-4، مقالات کامل و بهینه‌شده برای سئو در زمینه املاک ترکیه (به‌ویژه استانبول) به زبان فارسی تولید می‌کند.

## امکانات سیستم

### محتوای تولید شده
- ✅ مقالات 1200 تا 1800 کلمه‌ای
- ✅ لحن حرفه‌ای و قابل اعتماد
- ✅ نوشتار طبیعی و انسانی
- ✅ بهینه‌سازی کامل برای سئو

### عناصر سئو
- ✅ عنوان سئو (حداکثر 60 کاراکتر)
- ✅ اسلاگ URL (انگلیسی)
- ✅ توضیحات متا (حداکثر 160 کاراکتر)
- ✅ سرتیترهای ساختاریافته (H2, H3)
- ✅ بخش سوالات متداول (FAQ)
- ✅ پیشنهاد لینک‌های داخلی

### بخش‌های مقاله
1. مقدمه درباره منطقه
2. تحلیل روند قیمت (ماهانه و سالانه)
3. تحلیل سرمایه‌گذاری (بازده اجاره، ROI)
4. بهترین محله‌ها برای خرید
5. مقایسه خرید و اجاره
6. نکات قانونی و بازار
7. سوالات متداول (5 سوال)

## راه‌اندازی

### مرحله 1: دریافت کلید API

1. به سایت OpenAI بروید: https://platform.openai.com/
2. ثبت‌نام کنید یا وارد شوید
3. به بخش API Keys بروید
4. یک کلید جدید ایجاد کنید
5. کلید را کپی کنید

### مرحله 2: تنظیم متغیر محیطی

فایل `.env` در پوشه `backend` را باز کنید و خط زیر را اضافه کنید:

```env
OPENAI_API_KEY=sk-کلید-شما-اینجا
```

⚠️ **مهم:** کلید API را محرمانه نگه دارید.

### مرحله 3: راه‌اندازی سرور

```bash
cd backend
npm start
```

سرور باید بدون خطا اجرا شود.

## نحوه استفاده

### از طریق پنل ادمین

#### 1. ورود به پنل
- با حساب ادمین وارد شوید
- روی "Admin Panel" کلیک کنید
- تب "Blogs" را انتخاب کنید

#### 2. تولید بلاگ با هوش مصنوعی
- روی دکمه "🤖 Generate with AI" کلیک کنید
- فرم اطلاعات بازار را پر کنید

### فیلدهای الزامی

| فیلد | توضیحات | مثال |
|------|---------|------|
| **شهر (City)** | نام شهر | İstanbul |
| **منطقه (District)** | نام منطقه | Kadıköy |

### فیلدهای اختیاری

| فیلد | توضیحات | مثال |
|------|---------|------|
| **محله (Neighborhood)** | نام محله | Moda |
| **قیمت فروش متری** | میانگین قیمت فروش هر متر | 85000 |
| **قیمت اجاره** | میانگین قیمت اجاره ماهانه | 15000 |
| **تغییر ماهانه (%)** | درصد تغییر ماهانه | 2.5 |
| **تغییر سالانه (%)** | درصد تغییر سالانه | 25.3 |
| **سطح تقاضا** | پایین / متوسط / بالا | بالا |

### مثال عملی

**حالت پایه:**
```
شهر: İstanbul
منطقه: Beşiktaş
```
سیستم یک مقاله کامل درباره بازار املاک بشیکتاش تولید می‌کند.

**حالت کامل:**
```
شهر: İstanbul
منطقه: Kadıköy
محله: Moda
قیمت فروش متری: 85000
قیمت اجاره: 15000
تغییر ماهانه: 2.5%
تغییر سالانه: 25.3%
سطح تقاضا: بالا
انتشار خودکار: ✓
```
سیستم یک مقاله کامل و تخصصی با تحلیل دقیق بازار تولید می‌کند.

## زمان و هزینه

### زمان تولید
- یک مقاله: 30 تا 60 ثانیه
- چند مقاله: به‌ازای هر مقاله 1 دقیقه

### هزینه تقریبی
- هر مقاله: 0.02 تا 0.04 دلار
- 100 مقاله: 2 تا 4 دلار
- 1000 مقاله: 20 تا 40 دلار

💡 **نکته:** با توجه به کیفیت بالای محتوای تولیدی، این هزینه بسیار مقرون‌به‌صرفه است.

## خروجی سیستم

هر مقاله شامل:

1. **عنوان بهینه‌شده** - جذاب و سئو-دوست
2. **اسلاگ URL** - برای آدرس صفحه
3. **توضیحات متا** - برای نمایش در گوگل
4. **محتوای کامل HTML** - با فرمت‌بندی حرفه‌ای
5. **خلاصه مقاله** - برای نمایش در لیست
6. **بخش FAQ** - 5 سوال و جواب
7. **لینک‌های پیشنهادی** - برای لینک‌دهی داخلی

## نمایش در وب‌سایت

مقالات تولیدی به‌طور خودکار:
- ✅ در صفحه بلاگ نمایش داده می‌شوند
- ✅ سئوی کامل دارند
- ✅ بخش سوالات متداول دارند
- ✅ طراحی ریسپانسیو و زیبا
- ✅ پشتیبانی از راست‌چین فارسی

## نکات مهم

### قبل از انتشار
1. ✅ مقاله را بخوانید و بررسی کنید
2. ✅ در صورت نیاز ویرایش کنید
3. ✅ تصویر مناسب اضافه کنید
4. ✅ لینک‌های داخلی را پیاده‌سازی کنید

### برای بهترین نتایج
1. ✅ اطلاعات دقیق و به‌روز بدهید
2. ✅ هر هفته چند مقاله منتشر کنید
3. ✅ مناطق مختلف را پوشش دهید
4. ✅ عملکرد مقالات را رصد کنید

### امنیت
1. ⚠️ کلید API را محرمانه نگه دارید
2. ⚠️ هرگز در فایل‌های عمومی قرار ندهید
3. ⚠️ فقط ادمین‌ها دسترسی دارند

## رفع مشکلات رایج

### خطا: API key not found
**راه‌حل:** کلید API را در فایل `.env` تنظیم کنید.

### خطا: Rate limit exceeded
**راه‌حل:** چند دقیقه صبر کنید یا پلن OpenAI خود را ارتقا دهید.

### خطا: City and district required
**راه‌حل:** حتماً نام شهر و منطقه را وارد کنید.

### مقاله تولید می‌شود اما کیفیت پایین است
**راه‌حل:** اطلاعات بیشتری (قیمت، تغییرات، محله) وارد کنید.
